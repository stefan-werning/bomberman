- Es soll einen Server geben. Damit dieser erreichbar ist, soll eine nach außen öffentliche domain erstellt werden sobald man den server startet. mittels grok oder so, habe den genauen namen vergessen.
- Der server soll mit Node.js erstellt werden. Falls nötig, finde ein geeignetes framework.

- Öffnet jemand den Share Link, dann landet er in der Session.
- Auf dem Server Web Interface kann die Session gestartet werden.
